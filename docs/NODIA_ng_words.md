## カテゴリ5｜社内コミュニケーション注意ワード

> Slack・社内ドキュメントでも意識して避けるべき表現

| NG表現 | 理由 |
|---|---|
| あの人 / あいつ | 特定個人を指す曖昧な表現。固有名詞を使うこと |
| とりあえず | 方針の不明確さを示す。「まず〇〇を確認し、〜する」と具体化 |
| 普通に考えて | 価値観の押し付けになる場合がある |
| やっておいて | 依頼の主語・期限が不明確 |
| 〜な事 | 固いの印象 | 「〜なこと」 |
