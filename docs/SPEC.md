## 1. 解決したい課題

### 現状の課題

- 自律型人材を増やすことを掲げている企業は多いが、自律型か否かを適切に測るサーベイがない
- エンゲージメントサーベイがあるが代替はできない
- キャリア自律は結果論的に、自律的に働いて成果を出している人がいるとできているように見えるが客観的な指標がない
- 「大事である」ということは言われているが、経営や事業にとってプラスになるかの確証がないまま進めてしまう

### このプロダクトで解決すること

- 自社の「キャリアオーナーシップ」がどの程度発揮されているかを診断する
- データが貯まった場合は、同業界比較・同規模比較・同地域比較なども提供する

## 5. 診断設計

### 5-1. 診断の方向性

**方向性B（組織のC/O環境を測る）を採用**

人事担当者が、自社の組織としてのキャリアオーナーシップ推進環境を評価する診断。個人のキャリア自律度ではなく、組織・制度・文化・経営姿勢の状態を測定する。

### 5-2. 回答形式

**6件法（強制選択・中央値なし）**

```
1：全くない
2：ほとんどない
3：あまりない
4：ある程度ある
5：かなりある
6：十分にある
```

- 主観評価26問：全問必須回答
- 定量参考入力：任意・スキップ可（スキップしてもスコアに影響しない）

「5件法ではなく6件法を採用した理由：組織の状態を人事担当者というプロが評価する設計のため、中央値への集中（中心化傾向）を避け、診断精度を高めるため。」

### 5-3. 領域設計・配点・設問数

| 領域 | 優先度 | 設問数 | 重み |
|------|--------|--------|------|
| 経営・宣言 | 高 | 5問 | 20% |
| 管理職支援 | 高 | 5問 | 20% |
| 組織風土 | 高 | 5問 | 20% |
| 制度設計 | 中 | 4問 | 15% |
| 個人行動 | 中 | 4問 | 15% |
| データ把握 | 低 | 3問 | 10% |
| **合計** | | **26問** | **100%** |

優先度設定の背景：NODIAのコンサルティングおよびAIキャリアインタビュー提案において、個人よりも組織開発アプローチを優先するため、経営・管理職・組織風土を高優先度とした。

### 5-4. スコアリング計算方式

#### ステップ1：領域スコアの正規化（0〜100点）

```
領域スコア = （回答合計 − 設問数）÷（設問数 × 5）× 100
```

**計算例（全問3回答の場合）**

```
経営・宣言（5問、全問3回答）：
= （15 − 5）÷（5 × 5）× 100
= 10 ÷ 25 × 100
= 40点
```

全問3回答時の総合スコア：40点（「あまりない〜ある程度ある」の中間より低め）

#### ステップ2：重みづけ合算

```
総合スコア = Σ（各領域スコア × 重み）

= 経営・宣言スコア × 0.20
+ 管理職支援スコア × 0.20
+ 組織風土スコア × 0.20
+ 制度設計スコア × 0.15
+ 個人行動スコア × 0.15
+ データ把握スコア × 0.10
```

#### スキップ（定量参考入力）の扱い

- 定量参考入力はスコア計算に影響しない
- 主観評価26問は全問必須のためスキップ不可

### 5-5. 段階判定

| 段階 | スコア | 名称 | 状態の説明 |
|------|--------|------|-----------|
| 1 | 0〜19点 | 黎明期 | C/Oの概念がまだ組織に存在していない |
| 2 | 20〜39点 | 認知期 | 言葉は知っているが施策・文化がない |
| 3 | 40〜59点 | 理解期 | 施策は始まっているが行動変容に至っていない |
| 4 | 60〜79点 | 共感期 | 制度と文化が整い、一部の社員が動き始めている |
| 5 | 80〜100点 | 実践期 | 組織×上司×個人の三者が連動している状態 |

**段階設定の運用方針：**
- 初期は均等（20点刻み）でローンチ
- データが200件程度蓄積した時点で実態分布を分析し、非均等設計への調整を検討する

## 7. 画面詳細設計

### 7-1. トップ画面

- 診断の説明テキスト（目的・所要時間・活用方法）
- 基本情報入力フォーム（氏名・メール・職種・業界・従業員数・在籍企業）
- 「診断を開始する」ボタン
- 全必須項目入力後にボタンが活性化

### 7-2. 質問画面

#### 表示形式

- **1問ずつ表示**（1画面1問）
- 前の問に戻ることができる「戻る」ボタンを設置

#### 回答UIコンポーネント

**横並びボタン（6択）**

```
Q1. 経営層（CEO・CHROなど）が、キャリア自律の重要性を
    公式な場で繰り返し発信している

[ 1 ]  [ 2 ]  [ 3 ]  [ 4 ]  [ 5 ]  [ 6 ]
全くない                        十分にある

                              [ 次へ → ]
```

- 選択したボタンはハイライト表示
- 選択後に自動で次の問へ進む（もしくは「次へ」ボタンで進む）
- PCではキーボード（1〜6キー）でも回答可能にする

#### プログレス表示

**ステップバー（領域単位）**

```
[経営・宣言] ─── [管理職支援] ─── [組織風土] ─── [制度設計] ─── [個人行動] ─── [データ把握]
     ●                  ○                ○               ○               ○               ○
```

- 現在の領域をハイライト
- 完了した領域はチェック済みスタイル

#### 定量参考入力のUI

主観設問とは背景色・カードスタイルを変えて明確に区別する。

```
┌──────────────────────────────────────────┐
│ 📊 参考データ入力（任意・スキップ可）          │
│                                            │
│ 1on1の実施頻度を教えてください               │
│ ○ 週1回以上  ○ 月2〜3回                   │
│ ○ 月1回     ○ 実施していない               │
│                                            │
│ 実施率（わかる場合）: [      ] %            │
│                                            │
│ [ スキップする → ]    [ 入力して次へ → ]    │
└──────────────────────────────────────────┘
```
